# TURNABLE #

To get going create a ./build folder at the root of the directory for cmake, then run:
```
cmake -B build -S source/
```
Then after successful compilation you can user the ./remake.sh bash script


dependencies:
 - ncurses
 - sfml


possible libraries:

 - JUCE: A widely-used framework for developing audio applications and plugins, including support for MIDI.
 - RtMidi: A lightweight MIDI library for handling MIDI input and output.
 - PortAudio: Cross-platform library for audio input/output.
